# field-quotes

SKS Technologies quoting tool — part of the field apps suite at `fieldapps.verdalecres.xyz`.

A purpose-built quoting tool that lets SKS staff produce professional, client-ready quote documents quickly and consistently. Two modes:

- **Quick Quote** — small jobs, single area, fast output.
- **Large Quote** — full project submissions, multi-area, full proposal document.

Built inline with the rest of the field apps suite — same stack, same patterns, same look and feel as `/management`. Widgets and components port between apps.
